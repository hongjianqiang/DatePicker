一个包含日期和时间的纯JS日期控件，不依赖任何库或框架，方便集成。

一般使用

<input type="DatePicker">

配合Vue使用

输入的日期是：{{myDate}} 
<input type="DatePicker" v-model="myDate">
<span>输入的日期是：{{myDate}}</span>

<script>
    const app = new Vue({
        el: '#app',
        data: {
            myDate: ''
        }
    });
    window.app = app;
</script>
